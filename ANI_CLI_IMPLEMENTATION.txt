HOW TO USE

1. Install ani-cli (required):
   $ git clone https://github.com/pystardust/ani-cli.git
   $ cd ani-cli
   $ sudo make install

2. Install a video player (required):
   $ sudo apt install mpv    # Recommended

3. Launch Ani-GUI:
   $ bash run.sh

4. Search for anime:
   Type "Attack on Titan" in search box

5. Click result to select anime

6. Configure watching:
   • Quality: Select from dropdown
   • Episode: Leave at 0 for all, or enter number

7. Click "▶ Watch with ani-cli"

8. Enjoy watching!
